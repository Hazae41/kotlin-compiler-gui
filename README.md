# Kotlin-Compiler-GUI
Easily compile Kotlin classes (.kt) and run Kotlin scripts (.kts)

### Features

- Compile Kotlin classes

- Run Kotlin scripts

- Supports libraries for both compilation and running

- Do it as fast as the light with Drag and Drop support

- Clean & Minimalist UI
